# DJI Mavic 4 Pro Landing Page

A landing page for the DJI Mavic 4 Pro drone, built as Assignment 3 for the Front-End Development course.

## 📌 Project Overview

This project showcases a modern product landing page designed to promote and sell the DJI Mavic 4 Pro drone. The page highlights the product's features, compares it with competitors, displays customer reviews, and suggests related products.

## ✨ Features

* Navigation Bar
* Hero Section with product details and pricing
* Specifications Comparison Table
* Customer Reviews Section with sticky rating summary
* Related Products Section
* Footer with useful links and payment methods
* Modern and clean UI design

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Positioning (Fixed, Sticky, Absolute)

## 📂 Sections Included

1. Hero Section
2. Specifications Section
3. Reviews Section
4. Related Products Section
5. Navigation Bar
6. Footer

## 🎯 Assignment Objective

Create a product landing page that presents product information in a visually appealing way and encourages users to take action and purchase the product.

The project was implemented based on the provided assignment design and requirements.
